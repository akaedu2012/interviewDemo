# 分页功能修复说明

## 问题描述
用户反馈"分页功能没有实现"，实际情况是：
- ✅ 后端分页API已完整实现
- ✅ 前端分页逻辑已完成
- ✅ 分页UI组件已存在
- ❌ **问题根因**：默认 `pageSize=20`，而候选人总数只有15人，导致只有1页，分页控件不显示（条件：`totalPages > 1`）

## 解决方案

### 1. 调整默认 pageSize

**修改前**:
```typescript
const pageSize = parseInt(searchParams.get("pageSize") || "20", 10);
```

**修改后**:
```typescript
const pageSize = parseInt(searchParams.get("pageSize") || "10", 10);
```

**效果**:
- 默认每页显示 10 条记录
- 15 个候选人会分成 2 页（第1页10条，第2页5条）
- 分页控件会正常显示

### 2. 添加每页数量选择器 ⭐ 新增功能

创建 `PageSizeSelector` 组件，让用户可以自定义每页显示数量。

#### 组件代码
```tsx
// components/candidates/PageSizeSelector.tsx
export function PageSizeSelector({
  pageSize,
  onPageSizeChange,
}: PageSizeSelectorProps) {
  return (
    <div className="flex items-center gap-2">
      <span className="text-sm text-slate-400">每页显示</span>
      <Select value={String(pageSize)} onValueChange={handleChange}>
        <SelectTrigger className="w-[80px] h-8 border-cyan-500/30">
          <SelectValue />
        </SelectTrigger>
        <SelectContent>
          <SelectItem value="5">5 条</SelectItem>
          <SelectItem value="10">10 条</SelectItem>
          <SelectItem value="20">20 条</SelectItem>
          <SelectItem value="50">50 条</SelectItem>
        </SelectContent>
      </Select>
    </div>
  );
}
```

**功能特点**:
- ✅ 选项：5/10/20/50 条每页
- ✅ 科技风格样式（青色边框）
- ✅ 改变数量后自动重置到第1页
- ✅ URL参数同步

#### 集成到 CandidateList
```tsx
<div className="flex items-center gap-4">
  <CandidateSorter />
  <PageSizeSelector 
    pageSize={pageSize}
    onPageSizeChange={onPageSizeChange}
  />
  <div>统计信息...</div>
</div>
```

#### 添加处理函数（page.tsx）
```typescript
const handlePageSizeChange = (newPageSize: number) => {
  updateSearchParams({ pageSize: newPageSize, page: 1 }); // 重置到第1页
};
```

## 数据流程

### 用户改变每页数量
```
用户选择"5条/页"
  ↓
handlePageSizeChange(5)
  ↓
updateSearchParams({pageSize: 5, page: 1})
  ↓
URL更新: /?pageSize=5&page=1
  ↓
fetchCandidates()
  ↓
API: /api/candidates?pageSize=5&page=1
  ↓
返回前5条记录
  ↓
totalPages = Math.ceil(15 / 5) = 3
  ↓
显示分页控件（3页）
```

### 用户切换页码
```
用户点击"第2页"
  ↓
handlePageChange(2)
  ↓
updateSearchParams({page: 2})
  ↓
URL更新: /?pageSize=5&page=2
  ↓
fetchCandidates()
  ↓
API: /api/candidates?pageSize=5&page=2
  ↓
返回第6-10条记录
  ↓
显示第2页数据
```
