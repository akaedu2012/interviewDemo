## 设计系统规范

### 卡片样式
```tsx
className="glass-hover border border-cyan-500/20 ring-0"
```

### 卡片内边距
- 标准内边距: `px-6 pt-6 pb-6` 或 `px-6 py-6`
- Header: `px-6 pt-6 pb-4`
- Content: `px-6 pb-6`

### 标签样式
- 实色背景: `bg-cyan-600` / `bg-blue-600` / `bg-purple-600`
- 白色文字: `text-white`
- 字体加粗: `font-medium`
- 内边距: `px-3 py-1.5`
- 悬停效果: `hover:scale-105 transition-transform`

### 图标容器样式
```tsx
<div className="p-2 rounded-lg bg-gradient-to-br from-cyan-500/20 to-blue-500/20 border border-cyan-500/30">
  <Icon className="size-5 text-cyan-400" />
</div>
```

### 颜色主题
- **主标题**: `text-cyan-300`
- **副标题**: `text-slate-200`
- **标签**: `text-slate-400`
- **正文**: `text-slate-300`
- **青色**: `cyan-400`, `cyan-500`, `cyan-600`
- **蓝色**: `blue-400`, `blue-500`, `blue-600`
- **紫色**: `purple-400`, `purple-500`, `purple-600`

### 按钮样式
- 主要按钮: `bg-gradient-to-r from-cyan-500 to-blue-500 hover:from-cyan-600 hover:to-blue-600`
- 发光阴影: `shadow-lg shadow-cyan-500/20`
- 边框按钮: `border-cyan-500/30 hover:bg-cyan-500/10 hover:border-cyan-500/50`

## 用户体验改进

1. **视觉一致性**: 所有卡片使用统一的玻璃态效果和边框样式
2. **内容呼吸感**: 增加内边距，避免内容紧贴边框，提升阅读体验
3. **标签可读性**: 使用实色背景和白色文字，确保技能标签清晰可见
4. **图标识别度**: 彩色渐变图标容器，增强视觉层次
5. **交互反馈**: 按钮悬停效果和发光阴影，提升科技感
6. **加载体验**: 优化加载和错误状态的视觉呈现

## 技术细节

- 使用 Tailwind CSS 实用类
- 保持响应式设计（lg:col-span-1 等）
- 移除 Card 默认的 ring 边框（添加 `ring-0`）
- 使用 CSS 渐变和半透明效果
- 保持组件功能完整性，只优化视觉样式
