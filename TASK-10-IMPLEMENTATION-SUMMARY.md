## 技术亮点

### 1. URL 状态管理
所有筛选、排序、分页状态都存储在 URL query params 中：
```
/?page=1&pageSize=20&sortBy=score&sortOrder=desc&search=张三&skills=React,TypeScript&view=table
```

优势：
- 支持浏览器前进/后退
- 可分享的 URL
- 状态持久化

### 2. 搜索防抖
搜索输入使用 300ms 防抖，确保：
- 减少 API 请求次数
- 在 500ms 内响应（满足需求）
- 更好的用户体验

### 3. 响应式设计
- 卡片视图：1列（移动端）→ 2列（平板）→ 3列（桌面）
- 表格视图：横向滚动支持

### 4. 分数可视化
使用颜色区分分数范围：
- 🟢 绿色：80-100（优秀）
- 🟡 黄色：60-79（良好）
- 🔴 红色：0-59（需改进）

### 5. 状态颜色编码
使用不同颜色徽章表示候选人状态：
- 灰色：待筛选
- 蓝色：初筛通过
- 黄色：面试中
- 绿色：已录用
- 红色：已淘汰
