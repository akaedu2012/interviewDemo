## 🚀 技术栈

### 前端
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI 库**: shadcn/ui
- **图表**: Recharts
- **状态管理**: URL 参数 + React Hooks

### 后端
- **运行时**: Next.js API Routes
- **数据库**: SQLite + Drizzle ORM
- **AI 服务**: DeepSeek API (OpenAI SDK 兼容)
- **PDF 解析**: pdf-parse
- **验证**: Zod
