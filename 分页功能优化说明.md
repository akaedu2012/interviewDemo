# 候选人列表分页功能优化说明

## 优化概述
候选人列表已经具备完整的分页后端逻辑，本次优化重点是提升分页UI的视觉效果和用户体验，使其符合科技风格主题。

## 背景
- ✅ 后端分页API已实现（`/api/candidates` 支持 page、pageSize 参数）
- ✅ 前端分页逻辑已完成（`page.tsx` 中的 `handlePageChange` 函数）
- ✅ CandidateList 组件已有基础分页控件
- ❌ 分页UI样式需要优化，与科技主题不协调

## 优化内容

### 1. 分页控件容器优化

#### 玻璃态卡片包装
```tsx
<div className="glass-hover rounded-xl p-4 border border-cyan-500/20">
```
- **效果**: 玻璃态背景 + 青色边框
- **统一**: 与整体科技风格主题一致

#### 三栏布局
```
[页码信息] --------- [分页按钮组] --------- [快速跳转]
```

### 2. 页码信息显示

**左侧信息栏**:
```tsx
<div className="text-sm text-slate-400">
  第 <span className="text-cyan-300 font-medium">{page}</span> / 
  <span className="text-slate-300 font-medium">{totalPages}</span> 页
  <span className="mx-2">·</span>
  共 <span className="text-cyan-300 font-medium">{total}</span> 条记录
</div>
```
- **当前页/总页数**: 青色高亮显示
- **总记录数**: 青色高亮显示
- **分隔符**: 使用 · 符号

### 3. 分页按钮组优化

#### 上一页/下一页按钮
```tsx
className="border-cyan-500/30 hover:bg-cyan-500/10 hover:border-cyan-500/50 
           disabled:opacity-30 disabled:cursor-not-allowed"
```
- **边框**: 青色半透明
- **悬停**: 青色背景 + 加深边框
- **禁用**: 30% 不透明度 + 禁用光标

#### 页码按钮
**当前页（选中状态）**:
```tsx
className="bg-cyan-600 text-white border-cyan-500/50 hover:bg-cyan-700 
           font-medium shadow-lg shadow-cyan-500/30"
```
- **背景**: 青色实色 `bg-cyan-600`
- **文字**: 白色 + 加粗
- **阴影**: 青色发光效果
- **悬停**: 背景加深

**其他页码**:
```tsx
className="border-cyan-500/20 text-slate-300 hover:bg-cyan-500/10 
           hover:border-cyan-500/50 hover:text-cyan-300"
```
- **边框**: 淡青色
- **文字**: 浅灰色
- **悬停**: 半透明青色背景 + 边框加深 + 文字变青色

#### 页码按钮智能显示逻辑
- **总页数 ≤ 7**: 显示所有页码
- **当前页 ≤ 4**: 显示 1-7 页
- **当前页 ≥ 总页数-3**: 显示最后 7 页
- **其他情况**: 显示当前页前后各3页

### 4. 快速跳转功能 ⭐ 新增

**右侧快速跳转输入框**:
```tsx
<input
  type="number"
  min="1"
  max={totalPages}
  placeholder={String(page)}
  className="w-16 h-8 px-2 rounded-md border border-cyan-500/30 
             bg-slate-800/50 text-slate-200 text-center 
             focus:outline-none focus:border-cyan-500/50 
             focus:ring-1 focus:ring-cyan-500/30"
  onKeyDown={(e) => {
    if (e.key === "Enter") {
      const newPage = parseInt(target.value, 10);
      if (newPage >= 1 && newPage <= totalPages) {
        onPageChange(newPage);
      }
    }
  }}
/>
```

**功能特点**:
- ✅ 输入页码后按 Enter 键跳转
- ✅ 自动验证页码范围（1 到 totalPages）
- ✅ 跳转成功后清空输入框
- ✅ 科技风格样式（青色边框、深色背景、居中文字）

## 分页逻辑流程

### 数据流
```
用户操作 → handlePageChange(page) 
         → updateSearchParams({page}) 
         → URL更新 
         → fetchCandidates() 
         → API调用 /api/candidates?page=X
         → 返回分页数据
         → 更新candidates和total
         → 重新渲染列表和分页控件
```

### 参数说明
- **page**: 当前页码（从1开始）
- **pageSize**: 每页显示数量（默认20）
- **total**: 总记录数
- **totalPages**: 总页数（= Math.ceil(total / pageSize)）

## 技术细节

### 分页算法
```typescript
// 计算显示的页码范围（最多7个）
if (totalPages <= 7) {
  // 显示所有页码
  pageNumber = i + 1;
} else if (page <= 4) {
  // 当前页靠前，显示 1-7
  pageNumber = i + 1;
} else if (page >= totalPages - 3) {
  // 当前页靠后，显示最后7页
  pageNumber = totalPages - 6 + i;
} else {
  // 当前页居中，显示前后各3页
  pageNumber = page - 3 + i;
}
```

### 快速跳转验证
```typescript
const newPage = parseInt(target.value, 10);
if (newPage >= 1 && newPage <= totalPages) {
  onPageChange(newPage);
  target.value = ""; // 清空输入框
}
```
